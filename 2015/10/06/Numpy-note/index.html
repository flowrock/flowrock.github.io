array initialization

Numpy array object needs to be initialized from list object in python.

The dimension of Numpy array can be got using shape method

This will show the dimension in tuple (2,3)

Numpy array/matrix of zeros and ones can be created using np.zeros((5,5)) and np.ones([7,3]) respectively, note that the argument of zeros() and ones() can be either tuple or list.

division

The result of division by default is integer. To have float result without transforming number to float, we can import division from __future__ package beforehand under python2.7 version. The division will have float result (when result should be float) automatically.

indexing, slicing

Numpy is also 0 indexed, the indexing does not contain the index of the right bound.
Numpy use the method arange() to create an array in a range.

Numpy array does not create new array without explicitly statement.

Numpy array can only be copied using copy() method, the list copy method list2 = list1[:] in python.

To get specific rows/columns from matrix, need to put rows/columns indexes into list as the argument of indexing

transpose

Quick way to transpose is to use arr.T

Another way is to use arr.transpose()

The last way to transpose is to use arr.swapaxes(axis1, axis2)

Univeral array functions

Numpy array has many builtin funtions.

e.g.
np.sqrt(arr) calculates sqrt for every element in the array

np.exp(arr) calculates e^(each item in array)

np.random.randn(10) creates an array of 10 elements following normal distribution

np.maximum(A,B) needs array A and B in identical dimension and return the max of each same index of A and B

For boolean arrays, we can use arr.any() or arr.all() to see if there is at least one item that is true or all items are true in the array.

numpy.unique(arr) is to return all unique items in the array, like distinct in SQL.

numpy.in1d(arr1,arr2) is to return a boolean array showing membership of item of one-dimensional array arr1 in arr2.

Array input and output

To save the array data into .npy format on disk, use numpy.save('filename', array).

To load .npy file into workspace, use numpy.load('filename.npy').

To save multiple arrays into a zip file, use numpy.savez('filename.npz',x=arr1,y=arr2).

To load .npz file, still use archive = numpy.load('filename.npz') and show arr1 by archive['x'].

To save data into .txt file, use numpy.savetxt('filename.txt',arr,delimiter=',').

To load .txt file, use numpy.loadtxt('filename.txt',delimiter=',')
